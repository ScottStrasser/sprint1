Unit 1: Deisgning and Planning Web Pages

This unit has been completed and did not require a reflection.

Unit 2: Creating Web Pages with HTML5

Unit 2 has been an excellent introduction to HTML5. Being a CIS major, I enjoy working with code. This section was taught effectively, though maybe sometimes too slowly (which I think is a good problem to have). Overall the instructor has done a great job introducing me to the new concepts.

Unit 3: HTML Lists

This unit brought on more changes visually than any of the previous units. I learned important and useful concepts such as the box model. Specifically, this unit helped me understand how to:

  1. Stylize my webpage with font and color.

  2. Organize my webpage using the box model.

  3. Utilize IDs and Class to separate different elements on my page.

This unit was a very interesting introduction to graphics and the use of photo editing software. This was my first time using photoshop and it was more difficult than I'd expected. I also learned how to apply my knowledge of photo editing to my webpage. This unit helped me understand:

Unit 5: --
